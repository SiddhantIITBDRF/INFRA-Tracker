Export & Sync
How overrides work and how to push them to the live dashboard.
How the Admin Panel works:
All changes you make here are saved to your browser's localStorage as an "overrides" file. The dashboard reads this file and applies your edits on top of live Google Sheets data.

To push changes to the GitHub dashboard:
1. Click Export Overrides JSON above — this downloads overrides.json
2. Upload this file to your GitHub repo alongside index.html
3. The dashboard will automatically fetch and apply your overrides

Note: Overrides always win over Google Sheets data. To reset a project, use "Reset to Sheet" in the project editor.
